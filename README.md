# Expense Tracker with Analytics

A structured expense tracker built using Python, SQLite, and Streamlit.  
This application allows users to record income and expenses and view analytics such as totals, category-wise spending, and monthly summaries.

---

## Features
- Add income and expense transactions
- Persistent storage using SQLite
- Total income and total expense calculation
- Category-wise expense summary
- Monthly expense summary
- Simple and clean Streamlit UI

---

## Tech Stack
- **Python** – Application logic
- **SQLite** – Database for persistent storage
- **Streamlit** – Frontend interface
